# x86inject-tool

## Features
- **x86 and x64 support**
- **User and Kernel (Kernel-Bridge) support**
- **[c-comments](https://docs.microsoft.com/zh-cn/cpp/c-language/c-comments)**
- **Strings support ( "" or L"" or u8"" )**
- **import [WIP]**
